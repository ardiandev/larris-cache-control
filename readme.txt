== Description ==  
Larris Cache Control adds an admin panel switch to control cache clearing.  
Automatically clears theme pattern cache when changes are detected.  

== Installation ==  
1. Upload `larris-cache-control` to the `/wp-content/plugins/` directory.  
2. Activate the plugin through the 'Plugins' menu in WordPress.  
3. Go to **Cache Control** under Settings to enable or disable cache purging.
